# Despliegue en Vercel

Este proyecto está listo para ser desplegado en Vercel.

## Pasos para desplegar

1. Sube este repositorio a GitHub, GitLab o Bitbucket si aún no lo has hecho.
2. Ve a [https://vercel.com/import](https://vercel.com/import) e importa tu repositorio.
3. Vercel detectará automáticamente la configuración gracias al archivo `vercel.json`.
4. Haz clic en "Deploy" y espera a que termine el proceso.

¡Listo! Tu sitio estará disponible en una URL pública de Vercel.
